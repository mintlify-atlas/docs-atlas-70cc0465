## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js 18+** — [nodejs.org/en/download](https://nodejs.org/en/download)
- **Git** — [git-scm.com](https://git-scm.com)

## Get up and running

<Steps>
  <Step title="Clone the repository">
    Clone the Virasat source code from GitHub:

    ```bash
    git clone https://github.com/Avadhak47/NCERT.git
    cd NCERT
    ```
  </Step>

  <Step title="Install dependencies">
    Install all project dependencies using your preferred package manager:

    <CodeGroup>

    ```bash npm
    npm install
    ```

    ```bash yarn
    yarn install
    ```

    ```bash pnpm
    pnpm install
    ```

    </CodeGroup>

    This installs React 19, React Router, Framer Motion, D3, Tailwind CSS, and all other dependencies listed in `package.json`.
  </Step>

  <Step title="Start the development server">
    Launch the Vite development server:

    ```bash
    npm run dev
    ```

    The app opens at **[http://localhost:5173](http://localhost:5173)** by default.

    <Note>
      Virasat uses Vite 7 as its build tool, which provides near-instant server startup and fast hot-module replacement (HMR). Changes to any source file reflect in the browser immediately without a full page reload.
    </Note>
  </Step>

  <Step title="Open the interactive map">
    Navigate to [http://localhost:5173/explore](http://localhost:5173/explore).

    You'll see a D3-powered SVG map of India showing all 28 states and 8 union territories. Hover over any state to highlight it.

    <Tip>
      Use the regional filter tabs at the top of the map — **North**, **South**, **East**, **West**, **Central**, **Northeast**, and **Union Territories** — to focus on a specific part of India. The map zooms and highlights only the states in the selected region.
    </Tip>
  </Step>

  <Step title="Open a state's cultural detail panel">
    Click any state on the map to open its cultural detail panel on the right side of the screen.

    The panel includes the state's key monuments, festivals, traditional art forms, classical dances, folk music, and cuisine — all sourced from NCERT curriculum content. The URL updates to `/explore/:stateId`, so you can share or bookmark a specific state directly.
  </Step>

  <Step title="Explore the other sections">
    Use the glassmorphism navbar at the top to navigate the rest of the platform:

    | Route | Section | Description |
    |---|---|---|
    | `/timeline` | Timeline | India's history from prehistoric times to the modern era |
    | `/artforms` | Art Forms | Masonry gallery of paintings, handicrafts, monuments, and festivals |
    | `/games` | Games | 7 educational mini-games including map placement and culture trivia |
    | `/glossary` | Glossary | A–Z dictionary of cultural terms and heritage vocabulary |
    | `/acknowledgement` | Acknowledgements | Credits: NCERT, Survey of India, UNESCO Digital Museum |

    The navbar also contains a **language selector** (Globe icon) supporting 11 languages: English, Hindi, Bengali, Telugu, Marathi, Tamil, Urdu, Gujarati, Kannada, Malayalam, and Punjabi.
  </Step>
</Steps>

## Available scripts

Once the project is set up, you can use the following npm scripts:

```json package.json
{
  "scripts": {
    "dev": "vite",
    "build": "tsc -b && vite build",
    "preview": "vite preview",
    "lint": "eslint .",
    "convert-images": "node scripts/convert-to-webp.mjs",
    "upscale-images": "node scripts/upscale-images.mjs"
  }
}
```

| Script | Command | Description |
|---|---|---|
| Development server | `npm run dev` | Start the Vite dev server at `http://localhost:5173` |
| Production build | `npm run build` | Type-check with `tsc` then bundle with Vite |
| Preview build | `npm run preview` | Serve the production build locally for testing |
| Lint | `npm run lint` | Run ESLint across the entire codebase |
| Convert images | `npm run convert-images` | Convert source images to WebP format |
| Upscale images | `npm run upscale-images` | Upscale images using `sharp` |
