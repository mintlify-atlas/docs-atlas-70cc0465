The **Art Forms** page (`/artforms`) is a full cultural gallery aggregating heritage content from all states into a single browsable, filterable masonry grid. It includes paintings, performing arts, handicrafts, monuments, festivals, folk dances, 3D Hasta Mudras, and India's 8 classical dance forms.

## Filter categories

| Tab | Content |
|-----|---------|
| **All** | Everything — paintings, performing arts, handicrafts, monuments, festivals, folk dance |
| **Painting** | Traditional paintings from each state (Madhubani, Pattachitra, Kalamkari, etc.) |
| **Performing Arts** | Classical and folk dance forms, music, theatre |
| **Handicrafts** | Textiles, pottery, woodwork, metalwork, and other craft traditions |
| **Monuments** | Temples, forts, palaces, stupas, and UNESCO heritage sites |
| **Festivals** | Fairs and festivals from each state |
| **Folk Dance** | Regional folk dance forms distinct from classical traditions |

## Art form cards

Each card in the masonry grid shows:
- A heritage image (with automatic fallback if no local image is available)
- A type badge (e.g. `PAINTING`, `HANDICRAFTS`)
- The art form's title
- The state it belongs to (with a map pin icon)

Cards are sorted so items with real local images appear first. Items using fallback images from Unsplash sink to the bottom.

## Detail overlay

Click any card to open a split-panel overlay:

**Left panel** — full art form image with type badge overlay

**Right panel** — scrollable detail content:

## Hasta Mudras section

Scrolling below the masonry grid reveals the **Hasta Mudras 3D section** — 50 interactive WebGL models organized in a grid:

- **Asamyukta (Single Hand)** tab — 28 single-hand mudras
- **Samyukta (Two Hands)** tab — 22 two-hand mudras

Each model card shows the mudra name and a 3D viewer where you can:
- Drag to rotate the model
- Scroll/pinch to zoom
- Interact with the gesture from any angle

## Classical Dance Forms section

Below the mudras is a dedicated **Classical Dance Forms** grid showcasing the 8 principal classical dances of India — all originating from the traditions codified in the Natya Shastra:

1. Bharatanatyam (Tamil Nadu)
2. Kuchipudi (Andhra Pradesh)
3. Odissi (Odisha)
4. Kathak (Uttar Pradesh / Rajasthan)
5. Manipuri (Manipur)
6. Mohiniyattam (Kerala)
7. Kathakali (Kerala)
8. Sattriya (Assam)

Clicking any dance form card opens the same detailed overlay — these entries have the richest data including Navrasas, Asamyukta and Samyukta Hastamudra references, and artist portraits.

<Tip>
  The Classical Dance Forms section at the bottom of the Art Forms page has the most detailed overlays in the platform. They include Navrasas (9 expressions), gesture references for both single-hand and two-hand mudras, and artist portrait galleries.
</Tip>
